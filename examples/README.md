# Kavis UI Temel Örnekleri

Bu klasörde her biri tek bir özelliğe odaklanan bağımsız örnek crate'leri bulunur.

```bash
cargo run -p hello_world
cargo run -p input
cargo run -p system_monitor
cargo run -p window_title
cargo run -p app_assets
cargo run -p dialog_overlay
cargo run -p focus_trap
cargo run -p tooltip_top_edge
cargo run -p webview
```

Yeni örneklerde Türkçe Kavis UI adları tercih edilir: `Dugme`, `KokGorunum`, `SimgeAdi`, `Tema`, `EtkinTema`.
